# ⚡ Cyber-Nexus Dashboard

Un centre de commande personnel minimaliste et "Privacy-First" conçu pour centraliser mon workflow de cybersécurité et la gestion de mon infrastructure locale.

## 🚀 Fonctionnalités
- **Monitoring Infra :** Accès rapide à Proxmox, Pi-hole, NAS et routeur.
- **Tools Pentest :** Raccourcis directs vers TryHackMe, HTB et RootMe.
- **Productivité :** Pomodoro intégré, gestionnaire de tâches (To-Do) et agenda.
- **Design :** Interface Dark-Mode inspirée des terminaux (font: JetBrains Mono).

## 🛠️ Stack Technique
- **Frontend :** HTML5, CSS3 (Custom Variables & Grid Layout).
- **Logique :** Vanilla JavaScript (zéro dépendance externe).
- **Data :** Persistance locale via `localStorage`.

## ⚙️ Installation
1. Clone le dépôt : `git clone https://github.com/mtbeard/dashboard.git`
2. Ouvre `index.html` dans ton navigateur.
3. Modifie les adresses IP dans le fichier `index.html` pour correspondre à ton réseau.

## 🛡️ Disclaimer
Ce dashboard est un outil personnel. Attention à ne pas exposer publiquement vos adresses IP privées si vous partagez des captures d'écran.
